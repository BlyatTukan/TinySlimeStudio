Hellish Battle - 2.5D Retro FPS

Unity Assets

Project details

Overview
Hellish Battle is modeled on classics such as Doom (1993), Wolfenstein 3D (1992). It takes the best of the old classics, and adds elements from the latest FPS.
Render pipeline compatibility
This Package uses the Built-in Render Pipeline. If you want to use URP or HDRP you have to convert the project to other pipelines yourself. The code does not have its own materials, so there will be no major problems with it
Description
Welcome in Hellish Battle - 2.5D Retro FPS
Hellish Battle is a renewed version of the Game with the same title created by a 13-year-old boy who was just starting his adventure with programming and after 5 years he decided to renew his idea.
Technical details
Keyboard and Mouse: Support
Mobile (Touch Control): In the Next Update!
Controllers: Maybe in the future
160+ Black and White Level Texture at 32x32 resolution per 1 Units with Normal Texture
Game Level created with ProBuilder, but you can import ready-made level models and create them from Prefabs
Skybox Texture at 128x128 Resolution
